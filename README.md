1.cari dulu foto peluru, background bertema apa aja, orang yang naik bisa jet roket untuk sebagai si penembak dan musuhnya siapa aja oh iya sih soundtrack
2. kalo udah kekumpul semua saatnya bikin qoding yaitu player, enemy , background, soundtrack dan juga bullets
3.bikin kelompok yaitu class player(Gamesprite), enemy(Gamesprite),bullets(Gamesprite) sama class GameSprite(sprite.Sprite): itu sendiri
4.codingnya bisa secara bertahap misalnya backgroundnya dulu playernya enemynya bahkan bullets
5.jika sudah selesai codingnya makan dicoba dulu
6.nah cara mainnya pakai panah deket enter dibawahnya untuk gerakin player dan cara nembak pakai spasi
